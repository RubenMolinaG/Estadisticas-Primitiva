## Representación Gráfica de los Anillos.
Podemos dividir la matriz de la primitiva en 3 anillos:
- **Anillo Exterior**:
    - _[1..9], [40..49], [10..30], [19..39]._
- **Anillo Medio**:
    - _[11..18], [31..38], 21 y 28._
- **Anillo Interno**:
    - _[22..27]_
